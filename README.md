# 🌸🍵 Chai House 🏡✨

Welcome to **Chai House**, a fairy-tale inspired tea cottage website filled with warmth, flowers, and magical vibes.

## ✨ About The Project

Chai House is a one-page aesthetic website that combines:

- 🍵 Handcrafted Tea Experience
- 🌸 Floral Fairy-Tale Theme
- 🛍️ Boutique Corner
- 💌 Customer Reviews
- 🦋 Floating Decorative Elements
- ✨ Interactive Animations
- 📱 Responsive Design

The website is designed to create a cozy and dreamy atmosphere where visitors can explore teas, boutique products, and the story of Chai House.

## ✨ Features

### 🌸 Hero Section
A magical welcome section introducing visitors to Chai House.

### 🍵 Tea Menu
Featured tea selections including:

- Masala Chai
- Rose Tea
- Kashmiri Kahwa
- Chamomile Tea

### 🛍️ Boutique Corner
A collection of handmade gifts, journals, candles, and floral accessories.

### 💌 Customer Reviews
Automatic review slider showcasing visitor experiences.

### ✨ Interactive Effects
- Floating flowers
- Butterflies
- Sparkles
- Smooth animations
